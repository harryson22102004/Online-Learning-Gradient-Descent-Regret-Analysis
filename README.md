# Online Learning & Regret Analysis
Algorithms for learning from sequential data streams with dynamic environments.

### 🔬 Research Highlights
- **Online Gradient Descent**: Updates models incrementally as each new data point arrives.
- **Regret Bound Analysis**: Implements mathematical tracking of "Regret" — the difference between the model's loss and the optimal fixed predictor.
- **Optimization Focus**: Essential for real-time recommendation and bidding systems.

**Tech Stack:** Python, `NumPy`.
